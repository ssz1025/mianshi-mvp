# Cache Components

This skill provides comprehensive guidance on caching strategies for frontend components, with a focus on Next.js best practices.

## Key Features

- **Component Caching**: Offers strategies for caching React components to improve performance
- **Next.js Caching**: Provides guidance on Next.js specific caching mechanisms
- **Performance Optimization**: Helps optimize frontend performance through effective caching
- **Cache Invalidation**: Offers best practices for cache invalidation strategies
- **SSR/SSG Caching**: Provides guidance on caching for server-side rendering and static site generation

## Usage Guidelines

Invoke this skill when:
- You need help with component caching strategies
- You're optimizing frontend performance
- You're implementing caching in Next.js applications
- You need guidance on cache invalidation
- You're working with SSR/SSG caching
