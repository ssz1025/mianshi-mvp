# 代码审查 (Code Reviewer)

本技能提供全面的代码审查服务，帮助开发者提高代码质量、发现潜在问题并提供优化建议。

## 核心功能

- **代码质量评估**：分析代码的可读性、可维护性和性能
- **潜在问题检测**：识别可能的 bug、安全漏洞和逻辑错误
- **最佳实践建议**：提供符合行业标准的编码实践建议
- **性能优化**：识别性能瓶颈并提供优化方案
- **安全审查**：检测可能的安全漏洞和风险
- **代码风格检查**：确保代码符合项目的风格规范

## 使用场景

当用户需要以下帮助时，会触发此技能：
- 对代码进行质量审查
- 寻找代码中的潜在问题
- 优化代码性能
- 提高代码的可维护性
- 确保代码符合最佳实践
- 进行代码合并前的审查

## 审查范围

- 前端代码（JavaScript、TypeScript、React、Vue 等）
- 后端代码（Go、Python、Java、Node.js 等）
- 移动应用代码
- 数据库查询和存储过程
- 配置文件和构建脚本
