# Frontend Design

This skill provides comprehensive frontend design guidance based on industry best practices. It helps with:

## Key Features

- **UI/UX Design Principles**: Offers guidance on user interface and user experience design
- **Frontend Architecture**: Provides recommendations for frontend project structure and organization
- **Design Systems**: Helps implement consistent design systems across applications
- **Responsive Design**: Offers best practices for creating responsive and accessible interfaces
- **Performance Optimization**: Provides techniques for optimizing frontend performance

## Usage Guidelines

Invoke this skill when:
- You need help with designing user interfaces
- You're implementing a design system
- You want to improve frontend architecture
- You need guidance on responsive design
- You're optimizing frontend performance
