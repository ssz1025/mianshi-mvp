# Fullstack Developer

This skill provides comprehensive guidance for full-stack development, covering the entire application development lifecycle from frontend to backend, database, and deployment.

## Key Features

- **Frontend Development**: Offers guidance on modern frontend frameworks and libraries
- **Backend Development**: Provides best practices for server-side development
- **Database Design**: Helps with database schema design and optimization
- **API Development**: Offers guidance on RESTful and GraphQL API design
- **Deployment**: Provides strategies for application deployment and DevOps
- **DevOps**: Helps with CI/CD pipelines and infrastructure as code
- **Security**: Offers best practices for application security

## Usage Guidelines

Invoke this skill when:
- You need help with end-to-end application development
- You're building a full-stack application
- You need guidance on frontend-backend integration
- You're implementing database designs
- You're setting up deployment pipelines
- You need security best practices for your application
