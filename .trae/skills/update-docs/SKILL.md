# Update Docs (技术文档更新)

## 功能描述 (Description)

This skill helps update and maintain technical documentation, ensuring that documentation stays in sync with code changes and provides clear guidance for users and developers.

本技能帮助更新和维护技术文档，确保文档与代码变更保持同步，并为用户和开发者提供清晰的指导。

## 核心功能 (Key Features)

- **文档同步**：确保文档与代码变更保持同步
- **API 文档更新**：帮助更新 API 文档和接口说明
- **使用指南维护**：维护和更新用户使用指南
- **文档结构优化**：优化文档结构和组织
- **示例代码更新**：确保示例代码与最新 API 保持一致
- **文档质量检查**：检查文档的完整性和准确性

## 使用场景 (Usage Scenarios)

当用户需要以下帮助时，会触发此技能：
- 更新项目文档以反映代码变更
- 维护 API 文档和接口说明
- 优化文档结构和组织
- 确保示例代码与最新 API 保持一致
- 检查文档的完整性和准确性

## 最佳实践 (Best Practices)

- 保持文档与代码同步
- 使用清晰、一致的文档结构
- 提供完整的 API 文档
- 包含详细的使用示例
- 定期审查和更新文档
- 遵循项目的文档风格指南
